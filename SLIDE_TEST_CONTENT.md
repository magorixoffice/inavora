```

```

London

## Notes

- For slides requiring images/videos, make sure to have test files ready
- For external integrations (Miro, PowerPoint, Google Slides), ensure you have valid URLs
- Some slides may require specific settings to be configured in the editor
- Leaderboard slides work best after quiz slides have been completed
- Test both presenter and participant views for each slide type
